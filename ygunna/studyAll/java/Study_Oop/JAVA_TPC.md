### 5. 매서드는 변수다 (변수 vs 메서드의 관계)

#####  5-1. 변수와 메서드(method)

 변수: 데이터를 **한개의 형태로만** 저장 가능

 매서드: 동작을 한 후에 **한 개의 데이터(한개의 형태)만** 만들어 낸다.

*강의자의 논리: 한개의 데이터를 나타내므로 변수나 메서드는 유사하다.*

##### 5-2. 매서드의 매개변수 전달기법(parameter passing)

Call By Value(값 전달 기법)

Call By Reference(번지 전달 기법)
